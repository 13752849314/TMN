# A Matrix Transformer Network for Single Image Super-Resolution

## Environment

- pytorch>=2.0

### Quick start

```shell
git clone https://github.com/13752849314/MTN,git
cd MTN
```

### How to train

```shell
python train.py -opt ./config/MTN_x4.yml
```

### How to test

```shell
python test.py -opt ./config/MTN_x4.yml
```

Before training and testing, prepare the configuration file(*.yml)
